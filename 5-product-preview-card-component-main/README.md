## Overview

This project is a solution to the Frontend Mentor Product preview card component challenge. It's a responsive product card component that displays product information with a clean and modern design.

## Features

Fully Responsive Design: Adapts to all screen sizes (mobile, tablet, desktop)

Optimized Images: Uses <picture> element with responsive image sources

Interactive Elements: Hover effects on buttons and price display

Clean Typography: Uses custom fonts from Google Fonts

Accessible: Proper semantic HTML and ARIA attributes

Modern Styling: Gradient backgrounds and smooth transitions

Technologies Used :

HTML5: Semantic markup

Tailwind CSS: Utility-first CSS framework

CSS3: Custom styles and animations

Google Fonts: Montserrat and Fraunces fonts

SVG Icons: For cart icon

Design Principles

Development Workflow
